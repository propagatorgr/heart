# 🫀 Προσομοίωση Καρδιακού Ρυθμού (p5.js)

Η εφαρμογή **Προσομοίωση Καρδιακού Ρυθμού** είναι ένα εκπαιδευτικό διαδραστικό εργαλείο για τη διδασκαλία της Βιολογίας (Γυμνάσιο – Λύκειο).  
Στόχος της είναι η κατανόηση της λειτουργίας της καρδιάς μέσα από οπτικοποίηση και διερεύνηση διαφορετικών καταστάσεων ρυθμού.

Η εφαρμογή έχει σχεδιαστεί ώστε να είναι **παιδαγωγικά ασφαλής**, χωρίς ιατρικούς όρους ή σενάρια πανικού, και να χρησιμοποιείται εύκολα στην τάξη, σε εργαστήριο Η/Υ ή σε διαδραστικό πίνακα.

---

## 🎯 Εκπαιδευτικοί στόχοι

Οι μαθητές μπορούν να:

- συνδέσουν τον καρδιακό ρυθμό (bpm) με το χρονικό διάστημα μεταξύ παλμών (RR διάστημα)
- κατανοήσουν τη διαφορά μεταξύ:
  - φυσιολογικού ρυθμού
  - ταχυκαρδίας
  - αρρυθμίας
- διερευνήσουν πότε η καρδιά **δεν αντλεί αποτελεσματικά αίμα**
- παρατηρήσουν μια απλοποιημένη αναπαράσταση ΗΚΓ
- συζητήσουν τον ρόλο του συντονισμού στη φυσιολογία

---

## 🧩 Λειτουργίες της εφαρμογής

### 🔧 Χειριστήρια
- **Slider καρδιακού ρυθμού (bpm)**: 40–160 bpm
- **Αρρυθμία**: δημιουργεί ακανόνιστα RR διαστήματα
- **Ταχυκαρδία**: θέτει σταθερό υψηλό ρυθμό (150 bpm)
- **Ήχος καρδιάς**: ακουστική αναπαράσταση παλμών (ενεργοποιείται μετά από κλικ)
- **Επανόρθωση ρυθμού**: διαθέσιμη μόνο σε κρίσιμη κατάσταση
- **⛶ Πλήρης οθόνη / ⛶ Έξοδος από πλήρη οθόνη**

---

### 📊 Οπτικοποίηση
- Παλλόμενη καρδιά
- Απλοποιημένο ΗΚΓ
- Ενδείξεις:
  - Καρδιακός ρυθμός (bpm)
  - RR διάστημα (ms)
- Κατάσταση λειτουργίας:
  - Φυσιολογικός ρυθμός
  - Αρρυθμία
  - Ταχυκαρδία
  - 🚨 Ανεπαρκής άντληση αίματος

---

## ⚠️ Μηχανισμός επικινδυνότητας (εκπαιδευτικός)

Η ένδειξη **«Ανεπαρκής άντληση αίματος»** εμφανίζεται μόνο όταν συνυπάρχουν:

- ταχυκαρδία  
- αρρυθμία  
- πολύ μικρό RR διάστημα  

Δεν προσομοιώνεται καρδιακή ανακοπή.  
Το μοντέλο εστιάζει αποκλειστικά στη **λειτουργική απόδοση της άντλησης**.

---

## 🖥️ Fullscreen λειτουργία

Το κουμπί **⛶ Πλήρης οθόνη**:

- εμφανίζει την εφαρμογή σε πλήρη οθόνη
- κρύβει τα στοιχεία του browser
- αλλάζει αυτόματα σε **⛶ Έξοδος από πλήρη οθόνη**
- είναι ιδανικό για χρήση σε:
  - projector
  - διαδραστικό πίνακα
  - παρουσίαση στην τάξη

---

## 📱 Responsive σχεδιασμός

- Σε μεγάλες οθόνες: διάταξη **δύο στηλών**
  - προσομοίωση
  - χειριστήρια
- Σε μικρές οθόνες: διάταξη **μίας στήλης**
- Η προσαρμογή επιτυγχάνεται με απλό CSS (flexbox)

1. Βεβαιωθείτε ότι υπάρχουν τα αρχεία:
   - `index.html`
   - `sketch.js`
2. Ανοίξτε το `index.html` σε σύγχρονο browser (Chrome / Edge / Firefox)
3. Για ενεργοποίηση ήχου απαιτείται **ένα αρχικό κλικ** (πολιτική browser)
